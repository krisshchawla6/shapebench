# Context: Airfoil Design Optimization

## Parameter Space

Each airfoil is defined by N control points, where each point has 3 parameters:
- **radius_param**: Controls radial distance from center. Range: [-1.0, 1.0]
- **angle_param**: Controls angular offset from default position. Range: [-1.0, 1.0]  
- **edgy_param**: Controls corner sharpness (0=smooth, 1=sharp). Range: [-1.0, 1.0]

Action vector format: [r0, a0, e0, r1, a1, e1, ..., rN, aN, eN] (flat array of 3*N values)

## Objective

Maximize the reward (higher is better).

# Strategy: Incremental Improvement

Analyze the best-performing designs and make targeted improvements:
- Identify which parameters in top designs contribute most to performance
- Make small, systematic changes to improve upon the best known design
- Keep changes focused - modify 1-2 parameters significantly while keeping others similar to the best

# Analysis Task

Before proposing a new design, analyze the previous results:

1. **Best Design Analysis**: What was the best reward achieved? What parameters led to it?
2. **Pattern Recognition**: What patterns distinguish good designs from bad ones?
3. **Failure Analysis**: Why might low-performing designs have failed?
4. **Improvement Hypothesis**: Based on your analysis, what specific changes should improve performance?
