# Airfoil Design Context

## Parameter Space
Each airfoil is defined by N control points, where each point has 3 parameters:
- radius_param: Controls radial distance from center. Range: [-1.0, 1.0]
- angle_param: Controls angular offset from default position. Range: [-1.0, 1.0]  
- edgy_param: Controls corner sharpness (0=smooth, 1=sharp). Range: [-1.0, 1.0]

Action vector format: [r0, a0, e0, r1, a1, e1, ..., rN, aN, eN] (flat array of 3*N values)

# Airfoil Optimization Reference Card

## Global Heuristics
*   **The Radius-Distance Law:** Radius functions as polar distance from centroid. Maximize LE (CP0) and TE (CP2) radii while minimizing Upper (CP1) and Lower (CP3) radii to maximize chord and fineness ratio [iter 7, 8].
*   **The Slender Transition:** To exit the "stalled pebble" regime, use extreme high radii for CP0/CP2 (>0.9) and low radii for CP1/CP3 (<0.25) [iter 8].
*   **Chord Truncation:** Low TE Radius (CP2) "shortens" the chord, creating "Thimble" morphology [iter 7].
*   **The Scale Trap:** Low radii across all CPs create "miniature" bluff bodies with low Reynolds numbers [iter 4, 7].
*   **The Concavity Floor:** Radii below ~0.15 (specifically CP3) act as "gravity wells," creating concave "hooks" or "dents" rather than flat surfaces [iter 8].
*   **The Kutta Singularity:** TE Edgy [2][2] = 1.0 is mandatory for flow shedding; requires high CP2 Radius to prevent a "sharp wedge" truncation [iter 2, 5, 7, 8].
*   **Flow Attachment:** Steep recovery slopes (high CP1 Radius) trigger separation. Flattening via low CP1 Radius (~0.2) promotes laminar streamlines [iter 1, 8].

## Degree of Freedom Mapping

### CP0: Leading Edge (LE)
*   **Radius [0][0]:** Primary driver of forward chord extension. High values (>0.9) are essential for high-aspect-ratio slender bodies [iter 4, 7, 8].
*   **Angle [0][1]:** Controls vertical centering of the nose [iter 4].
*   **Edgy [0][2]:** -1.0 ensures a circular, C-infinity continuous stagnation point [iter 2, 5, 7].

### CP1: Upper Surface / Peak
*   **Radius [1][0]:** Controls "hump" fullness. Low values (~0.2) successfully thin the profile and break "pebble" morphology [iter 2, 7, 8].
*   **Angle [1][1]:** Lower values (~ -0.4) shift peak forward to ~30% chord [iter 1, 5, 8].
*   **Edgy [1][2]:** Low values ensure smooth transition over the peak [iter 4].

### CP2: Trailing Edge (TE)
*   **Radius [2][0]:** Primary driver of rearward chord extension. Must be near 1.0 to maximize total chord length and prevent truncation [iter 5, 7, 8].
*   **Angle [2][1]:** Aligns tail point; must be near 0.0 for horizontal chord alignment [iter 4].
*   **Edgy [2][2]:** 1.0 enforces a sharp geometric singularity for Kutta condition [iter 2, 5, 7, 8].

### CP3: Lower Surface
*   **Radius [3][0]:** Flattening/Camber control. Values ~0.25 flatten the surface; extreme low values (<0.1) create an abrupt concave "hook" or "chin" [iter 5, 7, 8].
*   **Angle [3][1]:** Positive values (~0.2) tuck the lower-rear inward. Combined with ultra-low radius, it creates localized concavity [iter 2, 4, 8].
*   **Edgy [3][2]:** High negative values (-0.9) maintain smooth lower surface continuity [iter 4].
