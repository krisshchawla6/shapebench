# Airfoil Design Context

## Parameter Space
Each airfoil is defined by N control points, where each point has 3 parameters:
- radius_param: Controls radial distance from center. Range: [-1.0, 1.0]
- angle_param: Controls angular offset from default position. Range: [-1.0, 1.0]  
- edgy_param: Controls corner sharpness (0=smooth, 1=sharp). Range: [-1.0, 1.0]

Action vector format: [r0, a0, e0, r1, a1, e1, ..., rN, aN, eN] (flat array of 3*N values)

# Airfoil Optimization Reference Card

## Global Heuristics
*   **Chord-Thickness Ratio:** Small horizontal distance between CP0 (LE) and CP2 (TE) relative to CP1/CP3 height results in "stubby/pebble" shapes with high pressure drag [iter 1].
*   **Flow Attachment:** Steep recovery slopes (high CP1 Radius + forward CP1 Angle) trigger massive flow separation [iter 1].
*   **Asymmetry Effects:** Extreme radius disparity between CP1 (Upper) and CP3 (Lower) produces "comma" or "hooked" profiles rather than flat bottoms [iter 1].
*   **Stretching:** Increasing X-axis distance between CP0 and CP2 is required to transition from "teardrop" to "aerodynamic" archetypes [iter 1].

## Degree of Freedom Mapping

### CP0: Leading Edge (LE)
*   **Radius [0][0]:** High values (>0.6) create blunt, bulbous noses. Disproportionately large if chord is short [iter 1].
*   **Angle [0][1]:** (Not specifically isolated in iter 1).
*   **Edgy [0][2]:** Low values (-0.95) produce smooth, circular stagnation points with continuous transitions [iter 1].

### CP1: Upper Surface / Peak
*   **Radius [1][0]:** Controls thickness/camber height. High values create high peaks; requires longitudinal stretching to maintain attached flow [iter 1].
*   **Angle [1][1]:** Lower values (~0.15) shift maximum thickness forward. Excessive forward bias increases slope severity toward the TE [iter 1].
*   **Edgy [1][2]:** (Not specifically isolated in iter 1).

### CP2: Trailing Edge (TE)
*   **Radius [2][0]:** (Not specifically isolated in iter 1).
*   **Angle [2][1]:** (Not specifically isolated in iter 1).
*   **Edgy [2][2]:** High values (~0.98) enforce a sharp point, satisfying the Kutta condition [iter 1].

### CP3: Lower Surface
*   **Radius [3][0]:** Low values (~0.22) pull the surface inward, creating a concave or "pinched" underbelly rather than a flat surface [iter 1].
