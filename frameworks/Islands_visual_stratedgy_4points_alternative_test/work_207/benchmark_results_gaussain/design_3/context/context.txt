# Airfoil Design Context

## Parameter Space
Each airfoil is defined by N control points, where each point has 3 parameters:
- radius_param: Controls radial distance from center. Range: [-1.0, 1.0]
- angle_param: Controls angular offset from default position. Range: [-1.0, 1.0]  
- edgy_param: Controls corner sharpness (0=smooth, 1=sharp). Range: [-1.0, 1.0]

Action vector format: [r0, a0, e0, r1, a1, e1, ..., rN, aN, eN] (flat array of 3*N values)

# Airfoil Optimization Reference Card

## General Design Rules
*   **Flow Direction**: Right-to-Left. CP0 is Leading Edge (LE); CP4 is Trailing Edge (TE) [iter 2].
*   **Drag Reduction**: Maximize chord-to-thickness ratio by maximizing TE radius and minimizing vertical radii (CP2/CP6) [iter 2].
*   **Lift Generation**: Introduce asymmetry (camber) by setting CP6 radius < CP2 radius [iter 2].
*   **Pressure Recovery**: Position maximum thickness at 25-30% of chord (via angle offsets) for stable subsonic flow [iter 2].
*   **Primary Axis**: Define via edginess extremes: LE (negative) vs. TE (positive) [iter 2].

## Degree of Freedom Mapping

| Control Point | Geometric Role | Optimal Range / Setting | Effect [Source] |
| :--- | :--- | :--- | :--- |
| **CP0 (Nose)** | Leading Edge | Rad: ~0.85 / Edg: -0.9 | Creates blunt, stable stagnation point; avoid bulbous over-radius [iter 2]. |
| **CP1 (Upper-Front)** | Transition | Intermediate | Smooths LE-to-Suction side transition; critical for avoiding pressure spikes [iter 2]. |
| **CP2 (Top)** | Suction Side | Rad: ~0.25 | Thins profile; reduces flow acceleration over the shoulder [iter 2]. |
| **CP3 (Upper-Back)** | Transition | Intermediate | Interpolates suction side to TE; affects flow separation at the rear [iter 2]. |
| **CP4 (Tail)** | Trailing Edge | Rad: 1.0 / Edg: 1.0 | Forces Kutta condition (sharp point); maximizes chord length [iter 2]. |
| **CP5 (Lower-Back)** | Transition | Intermediate | Interpolates TE to pressure side; sensitive to "kinks" or surface irregularities [iter 2]. |
| **CP6 (Bottom)** | Pressure Side | Rad: ~0.15 | Flattens lower surface relative to CP2 to create lift-generating camber [iter 2]. |
| **CP7 (Lower-Front)** | Transition | Intermediate | Critical for monotonic curvature; prevents early LE separation on the bottom [iter 2]. |
| **Angle Offsets** | Thickness Dist. | Small offsets | Shifts max thickness toward LE (ideal: first 30% of chord) [iter 2]. |
| **Edginess** | Surface Profile | -1.0 (Round) to 1.0 (Sharp) | Defines the "bluntness" or "sharpness" of the local curvature [iter 2]. |
