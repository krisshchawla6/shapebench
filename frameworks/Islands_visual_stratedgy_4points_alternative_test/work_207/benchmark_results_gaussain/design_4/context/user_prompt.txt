# Airfoil Design Context

## Parameter Space
Each airfoil is defined by N control points, where each point has 3 parameters:
- radius_param: Controls radial distance from center. Range: [-1.0, 1.0]
- angle_param: Controls angular offset from default position. Range: [-1.0, 1.0]  
- edgy_param: Controls corner sharpness (0=smooth, 1=sharp). Range: [-1.0, 1.0]

Action vector format: [r0, a0, e0, r1, a1, e1, ..., rN, aN, eN] (flat array of 3*N values)

# Airfoil Optimization Reference Card

## General Design Rules
*   **Flow Direction**: Right-to-Left. CP0=LE; CP4=TE [iter 2].
*   **Camber Trap**: Avoid radii < 0.2 on CP6; excessively low values create concave "bites" (high drag) rather than flat surfaces [iter 4].
*   **Spline Continuity**: Ensure gradual transitions in Edginess ($e$) between adjacent CPs to prevent "lumpiness" or surface oscillations [iter 4].
*   **TE Sharpness (Kutta)**: CP4 requires "support" from CP3 and CP5; if intermediate radii are too large or angles too wide, they "choke" the TE into a blunt shape [iter 4].
*   **Axis Alignment**: Keep angular offsets ($a$) small (< 0.1) to prevent the principal axis from tilting/rotating away from horizontal flow [iter 4].
*   **Pressure Recovery**: Position max thickness at 25-30% chord via conservative angle offsets [iter 2].

## Degree of Freedom Mapping

| Control Point | Geometric Role | Optimal Range / Setting | Effect [Source] |
| :--- | :--- | :--- | :--- |
| **CP0 (Nose)** | Leading Edge | Rad: ~0.85 / Edg: -0.9 | High negative edginess + offset creates a "beak"; keep offset near 0 for smooth stagnation [iter 4]. |
| **CP1 (U-Front)** | Transition | Gradual $e$ transition | Bridges LE to Suction side; sensitive to edginess jumps [iter 4]. |
| **CP2 (Top)** | Suction Side | Rad: > 0.25 / Offset: Low | Controls thickness; high offset creates localized "humps" rather than smooth crests [iter 4]. |
| **CP3 (U-Back)** | Transition | Radius < CP2 | Supports TE sharpness; must not "choke" CP4 extension [iter 4]. |
| **CP4 (Tail)** | Trailing Edge | Rad: 1.0 / Edg: 1.0 | Forces Kutta condition; ineffective if CP3/CP5 are too bulbous [iter 4]. |
| **CP5 (L-Back)** | Transition | Radius < CP6 | Mirrors CP3; critical for preventing blunt wake [iter 4]. |
| **CP6 (Bottom)** | Pressure Side | Rad: 0.2 - 0.3 | Lower bound for camber; values < 0.2 pull perimeter inward creating drag-heavy cavities [iter 4]. |
| **CP7 (L-Front)** | Transition | Gradual $e$ transition | Prevents early LE separation; must maintain monotonic curvature [iter 4]. |
| **Angle Offsets** | Thickness Dist. | $a < 0.1$ | Shifts thickness; excessive values distort chord length and rotate airfoil tilt [iter 4]. |
| **Edginess** | Surface Profile | $\Delta e < 0.5$ (adj. pts) | Defines curvature; rapid swings between CPs cause "wavy" non-monotonic surfaces [iter 4]. |
