# Airfoil Design Context

## Parameter Space
Each airfoil is defined by N control points, where each point has 3 parameters:
- radius_param: Controls radial distance from center. Range: [-1.0, 1.0]
- angle_param: Controls angular offset from default position. Range: [-1.0, 1.0]  
- edgy_param: Controls corner sharpness (0=smooth, 1=sharp). Range: [-1.0, 1.0]

Action vector format: [r0, a0, e0, r1, a1, e1, ..., rN, aN, eN] (flat array of 3*N values)

## Reference Card (Iteration 5)

### Global Design Rules
*   **Radial Positivity:** All $r_n > 0$ to prevent non-physical origin-crossing [iter 1].
*   **Radial Tapering Rule:** Avoid steep deltas ($>0.3$) between adjacent $r$ values. Rapid drops (e.g., $r_0=0.9 \rightarrow r_7=0.5 \rightarrow r_6=0.1$) cause **Radial Collapse**, creating concave "notches" that kill lift [iter 2, 5].
*   **Curvature Continuity:** Subsonic profiles must remain convex. Small $r_2, r_6$ relative to $r_0, r_4$ create "bean" shapes or "bites" that cause immediate flow separation [iter 3, 5].
*   **Eccentricity Context:** $e$ values are not absolute. A sharp $e_0 \rightarrow -1$ or blunt $e_4 \rightarrow +1$ will fail or distort if adjacent radii ($r_1, r_7$ or $r_3, r_5$) pull the spline too sharply [iter 5].
*   **Thickness-to-Chord Ratio:** Target $r_{vertical} \approx 12\text{--}15\%$ of total chord ($r_0 + r_4$) for optimal streamlining [iter 5].

### Degree of Freedom Map

| Parameter | Geometric Effect | Aerodynamic Implication |
| :--- | :--- | :--- |
| **$r_0$** | TE extension. | Defines chord; requires $r_1, r_7$ support to prevent "bulbous" tail distortion [iter 1, 5]. |
| **$e_0$** | TE Sharpness. | **Negative** ($\rightarrow -1$) for Kutta cusp. Only effective if $r_1, r_7$ taper gradually toward $r_0$ [iter 3, 5]. |
| **$r_2$** | Upper surface height. | Thickness control. Must be high enough to maintain convexity against $r_0, r_4$ [iter 2, 5]. |
| **$a_n$** | Surface bias. | Shifts control point influence; used to fine-tune max thickness location [iter 1, 2]. |
| **$r_4$** | LE extension. | Defines forward chord. High values require $r_3, r_5$ support to avoid "pointed" nose [iter 1, 5]. |
| **$e_4$** | LE Bluntness. | **Positive** ($\rightarrow +1$) rounds the nose. Requires adjacent $r$ support to prevent "lumpy" or jagged LE [iter 3, 5]. |
| **$r_6$** | Lower surface height. | Camber control. $r_6 < r_2$ creates lift, but $r_6 < 0.2$ often triggers "Radial Collapse" drag [iter 3, 5]. |
| **$r_{1,3,5,7}$** | Tapering support. | Must bridge $r_{even}$ neighbors via gradual transition (not just mean) to prevent structural collapse [iter 2, 5]. |
