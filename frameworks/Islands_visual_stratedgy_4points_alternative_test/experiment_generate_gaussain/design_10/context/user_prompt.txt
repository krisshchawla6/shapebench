# Airfoil Design Context

## Parameter Space
Each airfoil is defined by N control points, where each point has 3 parameters:
- radius_param: Controls radial distance from center. Range: [-1.0, 1.0]
- angle_param: Controls angular offset from default position. Range: [-1.0, 1.0]  
- edgy_param: Controls corner sharpness (0=smooth, 1=sharp). Range: [-1.0, 1.0]

Action vector format: [r0, a0, e0, r1, a1, e1, ..., rN, aN, eN] (flat array of 3*N values)

## Reference Card (Iteration 10)

### Global Design Rules
*   **Radial Positivity:** All $r_n > 0$ [iter 1].
*   **Laminar Delta Rule:** $\Delta r < 0.1$ between neighbors to prevent separation [iter 2, 6].
*   **The Scale Rule (Nugget Trap):** Total chord ($r_0 + r_4$) must strictly exceed 2.0. If $\le 2.0$, the body is a bluff cylinder with massive pressure drag, regardless of other parameters [iter 7, 10].
*   **The Synergy Rule:** Edginess ($e_n$) is secondary to radius ($r_n$). A sharp TE ($e_0 \rightarrow -1$) only works if $r_0 > 1.0$; otherwise, it creates a "flat-back" bluff body [iter 10].
*   **The Convexity Constraint:** Avoid the "Peanut" (waist) or "Flat Tire" (dent). $r_{even}$ values must not be local minima relative to neighbors $r_{odd}$ [iter 8, 10].
*   **Polar Thickness Logic:** $r_2, r_6$ are apexes. They must be supported by intermediate $r_{1,3,5,7}$ to form a convex hull [iter 8].
*   **The Pinch Technique:** For a sharp TE, $r_1, r_7$ must provide a monotonic taper toward $r_0$ [iter 6, 8].

### Degree of Freedom Map

| Parameter | Geometric Effect | Aerodynamic Implication |
| :--- | :--- | :--- |
| **$r_0$** | TE extension. | Primary chord builder. Must be $> 1.0$ to enable streamlining. If $r_0 \approx 1.0$, the aft remains bluff [iter 1, 10]. |
| **$e_0$** | TE Sharpness. | **Negative** ($\rightarrow -1$) for Kutta cusp. Requires $r_0 > 1.0$ to avoid creating a flat, drag-inducing rear face [iter 3, 10]. |
| **$r_2$** | Upper surface apex. | Primary thickness. Must be $\ge r_1, r_3$ to maintain convexity [iter 2, 8]. |
| **$a_n$** | Surface bias. | Shifts max thickness location. Forward bias ($a > 0$) helps delay separation on compact shapes [iter 1, 7, 10]. |
| **$r_4$** | LE extension. | Forward chord. Must be $> 1.0$ to define a distinct nose from the unit circle [iter 1, 10]. |
| **$e_4$** | LE Bluntness. | **Positive** ($\rightarrow +1$) rounds nose. $e \approx 0.5$ provides a stable stagnation zone [iter 3, 8]. |
| **$r_6$** | Lower surface apex. | Camber/Lift. $r_6 < r_2$ creates lift. Excessive reduction ($r_6 \ll 1.0$) causes a "Flat Tire" dent, disrupting flow [iter 3, 10]. |
| **$r_{1,3,5,7}$** | Tapering support. | Bridges $r_{even}$. Must be intermediate/monotonic to ensure smooth teardrop transition [iter 2, 8]. |
