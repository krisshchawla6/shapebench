## Reference Card (Iteration 4)

### Global Design Rules
*   **Radial Positivity:** All $r_n > 0$ to prevent non-physical origin-crossing [iter 1].
*   **Radial Gradient Continuity:** Large deltas between adjacent $r$ values (e.g., $r_0=1.0, r_1=0.5$) create a "pinched waist" or "mustache" effect. $r_{odd}$ must approximate the mean of $r_{even}$ neighbors for smooth transitions [iter 2, 4].
*   **Convexity & Scalloping:** Subsonic airfoils require convexity. Extreme low values ($r_6 < 0.2$) create concave "scallops" or "dents" that trap vortices and increase pressure drag [iter 3, 4].
*   **Eccentricity Polarity:** $e \rightarrow -1$ creates sharp, needle-like cusps (Kutta condition); $e \rightarrow +1$ creates broad, blunt bulbs (stall resistance) [iter 3, 4].
*   **Chord Alignment:** $r_4$ is the Leading Edge (LE); $r_0$ is the Trailing Edge (TE). Horizontal elongation ($r_0, r_4 \gg r_2, r_6$) defines slender, high-speed profiles [iter 3, 4].

### Degree of Freedom Map

| Parameter | Geometric Effect | Aerodynamic Implication |
| :--- | :--- | :--- |
| **$r_0$** | TE extension. | Defines chord length; high values create slender profiles [iter 1, 4]. |
| **$e_0$** | TE Sharpness. | **Negative** ($\rightarrow -1$) creates a needle-point cusp for clean flow exit (Kutta condition) [iter 3, 4]. |
| **$r_2$** | Upper surface height. | Controls thickness; low values flatten the "roof" to reduce thickness drag [iter 2, 4]. |
| **$a_2$** | Upper surface bias. | Shifts max thickness location to delay transition to turbulence [iter 1, 2]. |
| **$r_4$** | LE extension. | Defines forward chord; requires $r_3, r_5$ support to avoid "detached" look [iter 1, 4]. |
| **$e_4$** | LE Bluntness. | **Positive** ($\rightarrow +1$) rounds the nose; requires adjacent $r$ support to prevent "lumpy" transitions [iter 3, 4]. |
| **$r_6$** | Lower surface height. | Camber control; extreme lows ($<0.1$) cause "scalloping" and high drag [iter 3, 4]. |
| **$r_{1,3,5,7}$** | Structural support. | Must bridge $r_{even}$ neighbors; failure to provide "mid-point" support causes spline collapse/pinching [iter 2, 4]. |
