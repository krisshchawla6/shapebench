# Airfoil Design Context

## Parameter Space
Each airfoil is defined by N control points, where each point has 3 parameters:
- radius_param: Controls radial distance from center. Range: [-1.0, 1.0]
- angle_param: Controls angular offset from default position. Range: [-1.0, 1.0]  
- edgy_param: Controls corner sharpness (0=smooth, 1=sharp). Range: [-1.0, 1.0]

Action vector format: [r0, a0, e0, r1, a1, e1, ..., rN, aN, eN] (flat array of 3*N values)

## Reference Card (Iteration 7)

### Global Design Rules
*   **Radial Positivity:** All $r_n > 0$ to prevent origin-crossing [iter 1].
*   **Laminar Delta Rule:** Maintain $\Delta r < 0.1$ between all adjacent $r_n$ to prevent "lumpy" high-curvature zones and flow separation [iter 2, 6].
*   **Teardrop Topology:** Middle radii ($r_{1,2,3}$ and $r_{5,6,7}$) must be $\ge$ or provide a smooth bridge between end radii ($r_0, r_4$) to avoid "Peanut/Waist" shapes [iter 6].
*   **The Scale Rule:** Total chord ($r_0 + r_4$) must exceed 2.0. Reducing all radii to avoid penalties creates a "Nugget" (bluff body) with high pressure drag [iter 7].
*   **The Pinch Technique:** For a sharp Trailing Edge (TE), $r_1, r_7$ must be significantly smaller than $r_0$ to stretch the spline [iter 6].
*   **Chord-to-Thickness Ratio:** Target $r_{vertical} (r_2, r_6) \approx 12\text{--}15\%$ of total chord ($r_0 + r_4$). High thickness-to-chord ratios (~45%) result in "tilted bean" profiles rather than wings [iter 5, 7].

### Degree of Freedom Map

| Parameter | Geometric Effect | Aerodynamic Implication |
| :--- | :--- | :--- |
| **$r_0$** | TE extension. | Defines aft chord. Must be $> 1.0$ if $r_{2,6}$ are small to avoid "Scale Trap" [iter 1, 7]. |
| **$e_0$** | TE Sharpness. | **Negative** ($\rightarrow -1$) for Kutta cusp. Utility is lost if the tail is too short (flow detaches before cusp) [iter 3, 7]. |
| **$r_2$** | Upper surface height. | Primary thickness. High $r_2$ relative to $r_0$ creates a bluff body [iter 2, 7]. |
| **$a_n$** | Surface bias. | Shifts max thickness location. In compact/short shapes, causes "rotation/tilting" rather than volume shift [iter 1, 7]. |
| **$r_4$** | LE extension. | Defines forward chord. Requires $r_3, r_5$ support to avoid "pointed" nose [iter 1, 5]. |
| **$e_4$** | LE Bluntness. | **Positive** ($\rightarrow +1$) rounds nose. $e \approx 0.35$ provides smooth stagnation; $e > 0.8$ is "boxy" [iter 3, 7]. |
| **$r_6$** | Lower surface height. | Camber control. $r_6 < r_2$ creates lift. Must maintain $r_6 \ge r_0, r_4$ for structural bulk [iter 3, 6]. |
| **$r_{1,3,5,7}$** | Tapering support. | Bridges $r_{even}$ via $\Delta r < 0.1$ to maintain laminar flow [iter 2, 6]. |
